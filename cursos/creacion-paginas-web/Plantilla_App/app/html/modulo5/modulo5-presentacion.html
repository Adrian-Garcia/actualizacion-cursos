Hemos decidido hacer uso de Google Sites para crear el Sitio Web deseado, por lo que vimos en el módulo 4 sabemos que Google Sites es una herramienta gratuita que permite a cualquier persona crear Páginas Web de manera fácil y sencilla, y que no requiere conocimientos de programación en HTML.

Sin embargo no conocemos mucho de esta plataforma para poder crear nuestro Sitio Web con ella.

Bienvenido al quinto módulo:
“Creación de mi Página Web con Google Sites”

En este quinto módulo usted conocerá la herramienta de Google para la creación de Páginas Web, llamada Google Sites.

En el módulo anterior se enlistaron algunas maneras de publicar un Sitio Web, y  se presentó la herramienta de Google que sirve para crear y publicar Páginas Web; ahora, en este módulo, aprenderá a usar esta herramienta, Google Sites, para la creación de su Sitio Web.

Google no es sólo un buscador de información, también ofrece una serie de herramientas que permiten hacer muchas actividades en Internet, las cuales le serán de gran utilidad en la creación de su Sitio.

Google Sites  es la herramienta que permite crear y publicar un Sitio Web sin tener que contratar un servicio de hospedaje, ni usar un programa de transferencia de archivos, ni usar programación HTML,  ya que todo eso lo hará Google Sites por usted.

En este módulo usted conocerá la definición, las  ventajas de uso, los requerimientos de acceso, las secciones y los menús de Google Sites.



Al final de este módulo usted podrá:


Definir qué es, cuáles son sus ventajas y por dónde acceder a Google Sites.

Reconocer los requerimientos para entrar a Google Sites.

Identificar los elementos que componen la herramienta Google Sites.

Hacer uso de Google Sites para la creación de su Página Web.
