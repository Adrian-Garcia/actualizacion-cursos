Tema 2. Página y Sitio Web. Definición y características

Diferencia entre una Página y un Sitio Web

La diferencia entre una Página Web y un Sitio Web es que un Sitio Web está formado por varias Páginas Web y además pueden ofrecer servicios como: cuenta de correo, acceso a información especializada o un foro de discusión.

Página Web

Una Página Web es un documento básico de la Web, creado con lenguaje HTML (HyperText Markup Language, en español Lenguaje de Marcado de Hipertexto) donde el usuario de Internet navega entre sus contenidos accediendo a la información a través de *hipervínculos o enlaces que lo llevan a otras páginas con contenido relacionado. Una Página Web puede integrar imágenes, videos, fotos, salones de plática mejor conocidos como chat, y música, entre otros recursos. Un navegador, como por ejemplo, el Internet Explorer, es el programa en la computadora que le permite ver las Páginas Web de su preferencia.

* Los hipervínculos son representados por palabras, frases, botones o gráficos que "ligan" o "enlazan" a otra parte de la misma Página, a un documento o a una Página electrónica que puede estar en otra computadora de cualquier parte del mundo. De esta manera, se puede navegar de un Sitio a otro para acceder a la información que contiene Internet sin ninguna restricción. A un hipervínculo también se le llama liga o link

Sitio Web

Es el conjunto de Páginas Web que siguen determinada estructura y se encuentran organizadas de manera jerárquica. Dichas páginas se encuentran entrelazadas para mostrar al usuario información diversa sobre uno o varios temas en particular.

En un Sitio Web el usuario como dueño del contenido, tiene la libertad de publicar la información que desee, pero se recomienda que su Sitio Web tenga cierta estructura u orden que le permita al visitante del Sitio, entender la información de manera ordenada.

Cómo diferenciar entre una Página y un Sitio Web

Para explicar la diferencia entre un Sitio Web y una Página Web la compararemos con algo que todos conocemos… un centro comercial. A este centro comercial lo llamaremos Sitio Web. En él hay diferentes áreas de interés para cada una de las personas o familias que lo visitan diariamente, a estas personas, incluyéndolo, las llamaremos Usuarios de Internet.

Sabemos que cada persona tiene necesidades y gustos diferentes, algunas personas cuando visitan un Sitio Web buscan comprar, otras sólo van a pasear, existen otras que les encanta preguntar acerca de los productos, y otras más van a ofrecer productos o servicios.

A las tiendas que componen el centro comercial las llamaremos Páginas Web, éstas son los lugares donde las personas (Usuarios de Internet) entran de acuerdo a sus preferencias; cada tienda tiene un tema específico, como puede ser una tienda de discos, de juguetes, de muebles, o quizá puede ser un espacio tan amplio que tenga todo eso y para obtener cada uno de esos existen accesos a los que en la Web llamamos hipervínculos, que no son más que enlaces entre los productos (contenidos).
