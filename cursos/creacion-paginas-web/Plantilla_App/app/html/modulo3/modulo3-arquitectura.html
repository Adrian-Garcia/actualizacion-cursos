Tema 1. Arquitectura de la información

1. La arquitectura básica de una Página web

Definir la arquitectura de una Página Web es como definir la organización de los espacios de una casa.

Por ejemplo, generalmente una casa tiene una entrada, sala, comedor, cocina, baños, recamaras, y un área para lavar la ropa. Lo mismo pasa con una Página Web, la cual comúnmente cuenta  con una cabecera, barra de navegación, esta puede ser horizontal o vertical, un pie de página y el área de contenidos o materiales.



La arquitectura de un Sitio Web se refiere a la forma que tendrá el Sitio incluyendo secciones, y la manera de navegar de una sección a otra.

Como diseñador de un Sitio Web, definir la arquitectura de una Página Web, le va permitir tener mayor claridad sobre qué información se presentará a los visitantes.

2. Elementos de la arquitectura de una Página Web

Como diseñador de un Sitio Web, definir la arquitectura de una Página Web, le va permitir tener mayor claridad sobre qué información se presentará a los visitantes.

Existen 4 elementos básicos:

  • Cabecera
  • Barra de navegación
  • Contenido
  • Pie de página

Esta sección debe contener:

  1. El nombre del Sitio web.
  2. En caso de ser una empresa debe también incluir su logo.
  3. Buscador. Este es importante porque permite al visitante teclear lo que está buscando y que pueda encontrarlo de manera más rápida.

Generalmente la imagen o texto del nombre y/o logo de la empresa está ligada a la página principal y al darle clic le permite al usuario regresar a la página principal cuando está en alguna de las secciones del Sitio Web.

La barra de navegación también llamada menú se refiere al espacio donde habrá hipervínculos o enlaces a diferentes apartados del Sitio Web, los cuales se llaman secciones.

La barra de navegación o menú puede ser:

  1. Horizontal, se coloca debajo de la cabecera.
  2. Lateral, se coloca al costado de la página.
  3. Horizontal y lateral, es posible que existan los dos tipos de menú en un Sitio Web, pero sólo se recomienda cuando existe mucho contenido para publicar.

En un Sitio Web, es decir en un conjunto de Páginas Web relacionadas entre sí a través de hipervínculos …

Sí es necesaria

Una barra de navegación que es la que fungirá como vínculo entre las diferentes Páginas Web.

En una Página Web que no tiene relación con otras Páginas, Sitios o recursos…

No es necesaria

Una barra de navegación o menú.

3. Elementos de navegación

Para navegar (moverse) dentro de una Página Web, usted podrá usar:

  1. Las barra o menús de navegación
  2. Los hipervínculos o enlaces dentro del espacio de contenido

Es necesario que planee muy bien cómo utilizar los elementos de navegación para ayudar a los visitantes o usuarios del Sitio Web a navegar de manera sencilla y no confundirlo.
