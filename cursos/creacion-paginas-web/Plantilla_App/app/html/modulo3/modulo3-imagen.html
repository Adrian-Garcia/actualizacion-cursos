Tema 4. Imagen del Sitio Web

1. Elementos gráficos de un Sitio

Como aprendiste anteriormente, un tipo de contenido Web son las imágenes.

Los elementos gráficos de un Sitio Web están conformados por:

  • Imágenes
  • Botones
  • Íconos
  • Viñetas

Dentro de las imágenes se encuentran las ilustraciones, que se refieren a los dibujos tipo caricatura y las fotografías.


Los botones son elementos gráficos que indican una acción en concreto como: continuar, registrar, buscar, ir, etc. que permiten al usuario interactuar en el Sitio Web.


Los íconos se refieren a un grupo de elementos gráficos que sustituye un objeto a manera de representación de su significado.

Los íconos pueden tener cualquier diseño, mientras que se mantenga la unidad del significado.


Las viñetas se utilizan cuando hay contenidos que se enlistan, se utilizan al inicio de cada elemento de la lista (generalmente frase o párrafo).


2. Colores en Web

Los colores en Web están formados por colores primarios como son el Rojo, Verde y Azul, los cuales se representan de manera hexadecimal; es decir, que emplean un sistema de numeración de base 16.

Los colores se forman con tres pares de números, precedidos por el signo de gato.

Por ejemplo: #RRVVAA que es igual a un par de números rojos, un par de números verde y un par de números azules.

El color negro es #000000 es decir tres pares de Rojo-Verde-Azul y por el contrario el color blanco es #FFFFFF.

No es necesario que se aprenda este sistema de representación numérica ya que las herramientas para crear Páginas Web le permiten seleccionar el color de fondo, o color de letra de manera visible.

 

3. Características de las imágenes

Las imágenes pueden ser fotografías o ilustraciones.

Los archivos de tipo imagen tienen una extensión específica para poder ser publicado en Web, los cuales tienen algunas características que te mostramos a continuación:

BMP (Mapa de bits)

Este archivo se crea con el programa Paint de Windows.

Una de sus ventajas es que se puede abrir en cualquier programa de edición de imágenes, sin embargo, los archivos bajo esta extensión son muy pesados.

GIF(Graphics Interchange Format)

El uso de estos archivos es generalmente para Web, además de puede insertar en cualquier tipo de documento. Una de sus ventajas es que este formato es más ligero.

JPEG o JPG(Joint Photographic Experts Group)

Se utiliza principalmente en fotografías, pero se puede usar para cualquier tipo de imagen.

5. Obtención de imágenes

En este momento ya sabe qué tipo de imágenes necesita su Sitio, ya sean fotografías de sus productos u otro tipo de imágenes que expresen algo en especial en su Sitio.

Es importante usar imágenes de las cuales se tengan permiso para ser usadas de manera pública.


Existen algunos Sitios que le permiten usar sus fotos de manera gratuita, por ejemplo:

  • http://www.photogen.com/
  • http://www.foto-gratis.es/
  • http://www.pixelperfectdigital.com/free_stock_photos/


Recuerde siempre leer las políticas de descarga de cada Sitio de imágenes gratuitas.

Así mismo, si busca una imagen en http://es.wikipedia.org/wiki/Wikipedia:Portada  algunas tienen una licencia para ser usada de cualquier forma.

Dicha licencia se llama licencia de documentación libre GNU o la de Creative Commons donde se especifican los usos que se le pueden dar a las imágenes ahí publicadas.
